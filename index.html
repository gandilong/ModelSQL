ModelSQL

基于Apache的DbUtil而简化的SQL开发工具

到GitHub上查看项目 gandilong/ModelSQL

  • 下载beta 版ModelSQL.jar
  • 下载 源码TAR 包
  • View On GitHub

欢迎使用ModelSQL.

依赖包：

  • commons-lang-2.3.jar
  • commons-pool-1.6.jar
  • commons-dbcp-1.4.jar
  • commons-dbutils-1.5.jar
  • 相关数据库驱动包

    
	     DBExecutor dbe==new DBExecutor("com.mysql.jdbc.Driver","jdbc:mysql://localhost:3306/test","root","su","mysql");

注意事项：
当前版本是ModelSQL_V1.0.jar。
只限于MySQL和Oracle数据库适用。
实体主键字段只能是id(不分大小写)。

新增操作

dbe.insert(user);//插入到数据库
dbe.insertWithID(user);//自动生成ID值,并插入到数据库

删除操作

dbe.delete(User.class,102012);
或
dbe.delete(User.class,"flkwe-ew12-323jkl23");

修改操作

dbe.update(user);//实例中必须要有ID

查询操作

dbe.get(User.class,102012);//得到一条记录，并封装到User实例中。
dbe.get(User.class,"flkwe-ew12-323jkl23");

dbe.list(User.class);//查询所有user记录。
dbe.listDesc(User.class,"id");//查询所有user记录，并用id按降序排序
dbe.listAsc(User.class,"id,name");//查询所有user记录，并用id,name按升序排序


Condition condition=new Condition(User.class,new Page());//创建User类的条件实例并进行分页。
condition.like(name,"gandilong");
dbe.list(User.class,condition);
